Why to learn react ?
    React is a JS library used to build UI. when there was no react we used to build UI using jquerry and JS but react was developed by facebook for a purpose.
    We use react to build "complex frontend" like clicking a button changes/updates many things on a single page of facebook.

Why react was created ?
    State->JS  & UI-> DOM
    states are managed by JS and UI is managed by DOM.

    we saw the problem on a single facebook page, after reading msg the notification icon was still showing msg. it was due to the no conversation btw the states and UI. This problem was solved by react.


Topics to learn
    1. Core of react(state or UI manipulation, JSX)
    2. Component reusability- creating one button or card and using it many times.
    3. Props- property passing to Component.
    4. How to propagate changes(hooks)- useState, useEffect etc.
